git add . git commit -m "message" git push origin main git push -u origin main (to set upstream after that just git push will push to origin main) git init (to make a dir git initialized from local sys.) git remote add origin (to link the dir that was init in local sys. to git repo) git remote -v (to verify remote) git branch (to see the current branch) git branch -M main (to rename the current branch as main)
